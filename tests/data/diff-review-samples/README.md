# Diff Review Sample Set

These samples give the diff-analysis prompt a small committed baseline for local review work.

- `ui-workbench-layout.json`: cross-surface Svelte workbench change with UI and state impacts.
- `rust-diff-cache.json`: Rust-side diff payload and cache change with state and contract risk.

The samples are grounded in real repository patterns from Pi and are intended for manual spot checks,
prompt iteration, and future automated scoring.
